<h3><u>To run this program, do the following:</u></h3>

1. Open your terminal in the folder where these project files are stored

3. Type "streamlit run main.py"
4. A new window should open up in your default browser and the webpage should appear
5. To close the application, pull up your terminal again, and hit CTRL+C to terminate the program
